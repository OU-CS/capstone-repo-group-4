# Roam Frontend

The front end of Roam powered by Next.js

## Setup

Install dependencies 

```bash
npm install
```

Start local environment

```bash
npm run dev
```

## Deployment

Install Amplify CLI globally

```bash
npm install -g @aws-amplify/cli
```

Navigate into the frontend directory

```bash
cd frontend
```

Deploy to Amplify

```bash
npm run deploy
```
